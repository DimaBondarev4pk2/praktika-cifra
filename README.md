# Практика.Цифра

Полная версия дипломного проекта: веб-система для организации и контроля прохождения практики.

## Функции

- регистрация практикантов;
- регистрация руководителей практики с подтверждением администратором;
- авторизация и кабинеты трех ролей;
- задания, отчеты, дневник практики;
- загрузка и скачивание файлов;
- назначение руководителей;
- просмотр прогресса практикантов;
- восстановление пароля;
- светлая и темная темы, мобильная версия.

## Стек

PHP 8.1+, SQLite, PDO, HTML5, CSS3, JavaScript, nginx + PHP-FPM, SMTP.

## Локальный запуск

```powershell
php -d extension=pdo_sqlite -S localhost:8080 router.php
```

Открыть в браузере:

```text
http://localhost:8080
```

## Состав полной версии

В репозиторий включены файлы сайта, демонстрационная база данных, загруженные файлы и файл accounts.txt с тестовыми учетными записями.
